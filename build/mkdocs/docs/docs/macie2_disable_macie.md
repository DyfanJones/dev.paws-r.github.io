## Disables Amazon Macie and deletes all settings and resources for a Macie account

### Description

Disables Amazon Macie and deletes all settings and resources for a Macie
account.

### Usage

    macie2_disable_macie()

### Value

An empty list.

### Request syntax

    svc$disable_macie()
