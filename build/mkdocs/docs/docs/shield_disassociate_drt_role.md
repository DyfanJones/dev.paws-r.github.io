## Removes the Shield Response Team's (SRT) access to your Amazon Web Services account

### Description

Removes the Shield Response Team's (SRT) access to your Amazon Web
Services account.

### Usage

    shield_disassociate_drt_role()

### Value

An empty list.

### Request syntax

    svc$disassociate_drt_role()
