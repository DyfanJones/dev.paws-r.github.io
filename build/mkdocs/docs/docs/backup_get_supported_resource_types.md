## Returns the Amazon Web Services resource types supported by Backup

### Description

Returns the Amazon Web Services resource types supported by Backup.

### Usage

    backup_get_supported_resource_types()

### Value

A list with the following syntax:

    list(
      ResourceTypes = list(
        "string"
      )
    )
